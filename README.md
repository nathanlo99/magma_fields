# magma_fields

An implementation of Finite Fields in C++ as they (hopefully) appear in the Magma CAS

## Prerequisites

- CMake
- GMP (https://gmplib.org/)

## Build instructions

```
mkdir -p build
cd build
cmake ..
cd ../
cmake --build build
```

## To run the executable

```
build/magma_fields
```
